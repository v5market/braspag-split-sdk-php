# Instalando o SDK

Utilize o comando abaixo em seu terminal:

```bash
composer require v5market/braspag-split-sdk-php
```

Adicione o trecho abaixo em seu arquivo `composer.json`:

```json

```

E execute o comando `composer install` no terminal.
